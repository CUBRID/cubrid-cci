## HOW TO BUILD CUBRID CCI Driver
### CUBRID CCI Driver Source Build Guide
  * Requirement
    - GNU Developer Toolset 8 or higher (Linux)
    - MSVC 2017 (Windows)
    - CMake (Linux, Windows)

- Windows
  ```
  build)
  win\build.bat 
  
  help)
  win\build.bat /h
  ```
    
- Linux

  ```
  build)
  ./build.sh
  
  help)
  ./build.sh --help
  ```
